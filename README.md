# Projeto de Análise de Dados – Curso Hashtag

## Descrição
Este projeto foi desenvolvido durante as aulas do curso Hashtag Treinamentos, com o objetivo de praticar os principais conceitos de Análise de Dados utilizando Python.

## Objetivos
- Aplicar conceitos fundamentais de Análise de Dados
- Realizar análise exploratória dos dados
- Praticar manipulação de dados com Pandas

## Tecnologias Utilizadas
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Estrutura do Projeto
projeto_analise_dados/
- projeto_analise_dados.ipynb
- README.md
- .gitignore

## Como Executar
1. Abrir o arquivo projeto_analise_dados.ipynb no Jupyter Notebook ou no VS Code
2. Executar as células em ordem
